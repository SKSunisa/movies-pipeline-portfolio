# Movies Pipeline Portfolio

A React-based web application showcasing a complete data engineering pipeline for movie data, featuring multi-language support (Thai, English, Chinese).

üåê **Live Demo:** https://sksunisa.github.io/movies-pipeline-portfolio/

## Features

- **15 Pipeline Phases**: Complete documentation of the entire data pipeline from collection to visualization
- **Multi-language Support**: Full i18n support for Thai, English, and Chinese
- **Interactive Navigation**: Easy navigation between phases with dropdown menu and breadcrumbs
- **Responsive Design**: Mobile-friendly interface

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### `npm test`

Launches the test runner in the interactive watch mode.

### `npm run build`

Builds the app for production to the `build` folder.

### `npm run deploy`

Deploys the app to GitHub Pages.

## Tech Stack

- React 19 with Create React App
- react-i18next for internationalization
- React Router for navigation
- GitHub Pages for deployment
